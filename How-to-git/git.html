How to Git
Github远程仓库

第一步：安装Git

1.下载Git地址： 下载详细

2.打开Git目录下的git.cmd，输入 git --version 检查是否安装成功

第二步：配置用户

1.打开git.cmd，配置用户名和邮箱(下方以xiaoniezi为例)

git config --global user.name "xiaoniezi"
git config --global user.email "123456789@qq.com"
git config --global push.default matching

第三步：生成ssh

1.找到Git目录下的git.bash，进入.ssh目录 cd ~/.ssh

2.创建密钥 ssh-keygen -t rsa期间会有提示用户输入密钥的名字和密码，并二次确认密码，也可默认为空(下方以snow为例)

3.创建完成后查看目录文件 ls -al若出现snow和snow.pub两个文件即创建成功

第四步：添加公钥

1.注册github账户

2.进入用户的Settings的SSH and GPG keys设置选项

3.复制snow.pub公钥的内容，在github上添加SSH keys

4.同时在master分支下创建一个新的Repository(下方以test为例)

第五步：检测密钥

1.在git.bash主目录下输入ssh-add -l若出现下方的提示继续以下步骤，否则跳到第六步

Could not open a connection to your authentication agent.

2.输入以下命令

eval "$(ssh-agent -s)"
ssh-add ~/.ssh/snow

3.尝试连接github ssh -T git@github.com出现以下提示则成功

Hi xiaoniezi!You're successfully authenticated, but Github does not provide shell access.

第六步：远程仓库

1.继续使用git.bash，进入你想与github远程的目录cd d:/github/snow

2.初始化：git init

3.添加当前目录的所有文件git add .

4.提交文件git commit -m "mark something"

5.远程githubgit remote add origin git@github.com:xiaoniezi/test.git

6.push到test项目下git push -u origin master

7.到github的test上即可看到提交的文件

PS：如若错误，找github官方帮助文档：click me
